# Railway Ticket System

## Steps to comile and build
1. Compile the protobuf
```shell
make compile-proto
```

2. Build grpc server
```shell
make build-server
```

3. Run grpc client
```shell
make build-client
```

## Steps to run server and client
1. Run server
```shell
make run-server
```
2. Run client
```shell
make run-client
```

## Run test cases
1. Run server test cases
```shell
make test-server
```
